# About
Simple Tracker for Recording and Identifying Visual Expenses : Strive is a financial transaction and categorical limit tracker, with a visualization front-end built for students to gain insight into their spending habits.

# Features
- Financial transaction tracking
- Categorical spending limits
- Various visualization tools for data
- Persistent data and export capability
